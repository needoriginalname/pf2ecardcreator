# Support This Project

NeedOriginalPf2eTools is a free, open-source project maintained by volunteers who are passionate about making Pathfinder 2e more accessible to everyone. If you find this tool helpful, here are several ways you can support its development:

## 💙 Non-Financial Support

You don't need to spend money to help! Here are meaningful ways to contribute:

### 📣 Spread the Word

- Share NeedOriginalPf2eTools with your gaming group
- Recommend it to fellow PF2e players
- Post about it on social media (Reddit, Discord, Twitter, etc.)
- Write a review or testimonial

### 🐛 Report Issues & Suggest Features

- Found a bug? [Report it on GitHub](https://github.com/needoriginalname/pf2ecardcreator/issues)
- Have a feature idea? [Create a feature request](https://github.com/needoriginalname/pf2ecardcreator/issues)
- Quality feedback helps us improve!

### 💻 Contribute Code

We welcome contributions from developers of all skill levels:

- Fork the [repository](https://github.com/needoriginalname/pf2ecardcreator)
- Make your improvements
- Submit a pull request
- Help us fix bugs and add features

### 📖 Improve Documentation

- Help us write better guides
- Improve this documentation
- Create tutorials for new users
- Fix typos and clarify explanations

### 🤝 Join Our Community

- Participate in GitHub discussions
- Help answer questions from other users
- Share your character builds created with our tool
- Give constructive feedback

### How to Get Started Contributing

1. **Review** the [GitHub repository](https://github.com/needoriginalname/pf2ecardcreator)

3. **Fork** the repository and make your changes
4. **Submit** a pull request with a clear description
